# CTS Java Full Stack Engineer (Java FSE) Training

This repository contains my practice programs, hands-on exercises, and assignments completed as part of the Cognizant Java Full Stack Engineer (Java FSE) training program.

The repository covers core Java, Object-Oriented Programming, Design Patterns, Data Structures, Maven, Spring Framework, Spring Boot, Spring Data JPA, Hibernate, SQL, PL/SQL, JUnit, Mockito, and other concepts learned throughout the training.

The primary objective of this repository is to strengthen my understanding of Java Full Stack development through practical implementation and continuous learning.
